### Configuración de Contactship en Make

Para integrar Contactship con Make, sigue estos pasos:

1. **Crear una API Key**: 
   - Inicia sesión en tu cuenta de Contactship.
   - Navega a la sección de API y genera una nueva API key.

2. **Instalar el Módulo de Contactship**:
   - Ve a la plataforma de Make.
   - Busca el módulo de Contactship e instálalo.

3. **Conectar el Módulo**:
   - Abre el módulo de Contactship en Make.
   - Ingresa tu API key para establecer la conexión.

4. **Configurar el Módulo de Llamada AI**:
   - Completa los campos requeridos:
     - **Desde Número de Teléfono**: Tu número de contacto.
     - **Número de Teléfono a Llamar**: El número del destinatario.
     - **Nombre Completo del Usuario**: El nombre del usuario que realiza la llamada.
     - **Correo Electrónico del Usuario**: La dirección de correo del usuario.
     - **País del Usuario**: El país del usuario.
     - **Variables Personalizadas**: Agrega cualquier dato adicional como pares clave-valor.

5. **Automatizar Llamadas**:
   - Una vez configurado, puedes activar llamadas como parte de tu flujo de trabajo en Make.

Esta integración te permitirá optimizar tu proceso de llamadas y mejorar tus campañas.
