### Configurando Contactship en Make

Para integrar Contactship con Make, sigue estos sencillos pasos:

<AccordionGroup>
  <Accordion title="1. Obtén tu Clave API">
    - Inicia sesión en tu cuenta de Contactship.
    - Navega a la sección de **Claves API** en el menú de **Organizaciones**.
    - Haz clic en **Generar Clave** para crear tu clave API.
    - **Importante**: La clave API solo se mostrará una vez por razones de seguridad. Asegúrate de guardarla de forma segura.
    <Frame>
      ![Claves API](/images/apikeys.png)
    </Frame>
  </Accordion>

  <Accordion title="2. Instala Contactship en Make">
    - Ve a la plataforma Make.
    - Usa el siguiente enlace de invitación para instalar el módulo de Contactship: [Instalar Contactship](https://us1.make.com/app/invite/0fbc06d185316aa84bf499d6cfdfd87b).
    - Haz clic en **Instalar**, y el módulo se añadirá a tus escenarios.
    <Frame>
      ![Instalar Módulo](/images/SCR-20240831-cclb.png)
    </Frame>
  </Accordion>

  <Accordion title="3. Encuentra el Módulo de Contactship en Tus Escenarios">
    - Después de la instalación, abre tus escenarios en Make.
    - Busca el módulo de **Contactship** en la lista de módulos disponibles.
    - Arrastra y suelta el módulo en tu escenario para comenzar a usarlo.
    <Frame>
      ![Encontrar Módulo](/images/SCR-20240831-byzs.png)
    </Frame>
  </Accordion>

  <Accordion title="4. Conecta el Cliente de Contactship en Make">
    - Abre el módulo de Contactship en Make.
    - Ingresa tu clave API para establecer la conexión.
    
    <Frame>
      ![Ingresar Clave API](/images/SCR-20240831-bzpy.png)
    </Frame>
    
    - Localiza el campo etiquetado como **Clave API** en la configuración del módulo.
    - Pega tu clave API en este campo para conectarte con éxito.
    
    <Frame>
      ![Campo Clave API](/images/SCR-20240831-bzwn.png)
    </Frame>
    
    - ¡Ahora estás listo para usar las funciones de Contactship en tus escenarios!
  </Accordion>

  <Accordion title="5. Configura los Ajustes de Llamada Telefónica">
    - En el módulo de Contactship, completa los campos requeridos:
      - **Número de Teléfono Desde**: Tu número de teléfono.
      - **Número de Teléfono a Llamar**: El número de teléfono del contacto.
      - **Nombre Completo del Usuario**: El nombre del contacto.
      - **Correo Electrónico del Usuario**: La dirección de correo electrónico del contacto.
      - **País del Usuario**: El país del contacto.
      - **Variables Personalizadas**: Agrega cualquier dato adicional como pares clave-valor sobre el contacto.
    <Frame>
      ![Ajustes de Llamada Telefónica](/images/SCR-20240831-cada.png)
    </Frame>
  </Accordion>

  <Accordion title="6. Realiza Tu Primera Llamada">
    - Una vez que todo esté configurado, puedes iniciar una llamada utilizando los ajustes configurados.
    - Haz clic en el botón **Hacer Llamada Telefónica AI** para comenzar la llamada.
    <Frame>
      ![Hacer Llamada](/images/SCR-20240831-bzwn.png)
    </Frame>
  </Accordion>

  <Accordion title="7. Revisa los Detalles de la Llamada">
    - Después de la llamada, tienes dos opciones para revisar los detalles y registros para asegurarte de que todo funcionó como se esperaba:
      1. **Desde la Plataforma de Contactship**:
         - Navega a la sección de **Llamadas API** en tu panel de Contactship.
         - Aquí puedes revisar todas las [llamadas API](https://app.contactship.ai/es/organization/api-calls) junto con sus transcripciones, grabaciones de audio, detalles de la llamada y análisis.
         <Frame>
           ![Revisar Llamada en Plataforma](/images/SCR-20240905-bgkz.png)
         </Frame>
      2. **Por Webhook**:
         - Puedes configurar un webhook para recibir detalles de la llamada después de que se realice.
         - Consulta la [Documentación de Webhooks](https://docs.contactship.ai/webhooks) para obtener instrucciones sobre cómo configurar y usar webhooks.
  </Accordion>
</AccordionGroup>

### Consejos para una Integración Exitosa

- **Conexión Segura**: Mantén tu clave API segura y no la compartas públicamente.
- **Documentación**: Consulta la [documentación de la API](https://docs.contactship.ai/es/api-reference) para obtener detalles sobre cómo usar la API en tus integraciones.
- **Soporte**: Si tienes preguntas o necesitas asistencia, no dudes en contactar a nuestro equipo de soporte.
