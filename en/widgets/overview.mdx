### Web Call Widget

To enable web calls, you can:

1. **Embed the Widget**: Add the provided widget code to your website or platform.
2. **Configure the Widget**: Customize the widget settings to match your branding and requirements.
3. **Enable Web Calls**: Allow your customers/users/leads to make calls directly from the web interface.
