### API Integration

Automate workflows by integrating with our API:

1. **API Documentation**: Refer to our [API documentation](https://mintlify.com/docs/api-playground/openapi/setup) for detailed instructions on how to use the API.
2. **Authentication**: All API endpoints are authenticated using Bearer tokens.
3. **Automate Tasks**: Use the API to automate tasks such as making calls, retrieving call data, and managing agents.
